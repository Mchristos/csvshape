# csvshape
A simple CLI to show csv shapes 


## Usage 

Print the shape of all csvs in currect directory
```
> csvshape 
./a-file1.csv: (1341,195)
./a-file2.csv: (440,195)
./b-file3.csv: (6438,195)
./b-file4.csv: (2100,195)
```

Using a wildcard

```
> csvshape a-*
./a-file1.csv: (1341,195)
./a-file2.csv: (440,195)
```
